## API

| 属性        | 说明     | 类型               | 是否必须  | 默认值   |
| --------- | ------ | ---------------- | ----- | ----- |
| text      | 显示的文本  | string           | true  | -     |
| textType  | 文本类型   | text,link,strong | false | text  |
| href      | 跳转链接   | string           | false | -     |
| thousands | 添加千分位符 | boolean          | false | false | 
|style | CSS内联属性 | CSSProperties | false |-|
|className | class Class | string | false | -|
