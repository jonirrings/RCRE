## 没有输入展示的文字
使用`placeholder`属性来让输入框在没有输入的时候显示提示

```json
{
    "body": [
        {
            "type": "container",
            "model": "form",
            "children": [
                {
                    "type": "input",
                    "name": "placeholder",
                    "placeholder": "请输入文字"
                }
            ]
        }
    ]
}
```
