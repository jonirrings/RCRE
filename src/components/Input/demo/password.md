## 密码输入框

设置`inputType`属性来让输入框变成一个密码输入框

```json
{
    "body": [
        {
            "type": "container",
            "model": "form",
            "children": [
                {
                    "type": "input",
                    "name": "password",
                    "inputType": "password"
                }
            ]
        }
    ]
}
```
