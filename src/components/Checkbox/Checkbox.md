## API

| 属性        | 说明     | 类型               | 是否必须  | 默认值   |
| --------- | ------ | ---------------- | ----- | ----- |
| name | Checkbox的数据模型Key | string | true | - | 
| defaultChecked | 初始是否选中 | boolean | false | false | 
| disabled | 是否禁用 |  boolean | false | false |
| text | 显示的文字 | string | false | - |
| className | CSS class | string | false | - |
| style | 内联CSS属性 | CSSProperties | false | - |
